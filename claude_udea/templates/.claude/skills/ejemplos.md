# /ejemplos [tema]

El estudiante quiere ejemplos prácticos de un tema.

## Instrucciones

1. Primero busca en las transcripciones si el profesor ya dio ejemplos de ese tema en clase. Si los dio, preséntalos primero:

```
## Ejemplos del profesor
**Fuente:** [archivo.vtt | ~min XX]

[Ejemplo tal como lo explicó el profesor]
```

2. Luego, si el estudiante quiere más, ofrece ejemplos adicionales propios. Estos deben ser:
   - **Prácticos**: que se puedan ejecutar o aplicar
   - **Claros**: sin ambigüedad, paso a paso
   - **Relevantes**: alineados con el nivel y enfoque de la clase
   - **Concisos**: sin explicaciones innecesarias

3. Si el tema involucra código, da código funcional con comentarios breves.

4. Si el tema es teórico, usa analogías simples o casos reales.

5. Estructura:

```
## Ejemplos adicionales

### Ejemplo 1: [título descriptivo]
[Ejemplo]

### Ejemplo 2: [título descriptivo]
[Ejemplo]
```

6. Máximo 3 ejemplos a menos que el estudiante pida más.

7. Cada ejemplo debe conectarse con lo visto en clase: "Esto aplica a lo que vieron en [clase X] sobre [tema Y]."
